FortuneTale - Personal Asset Management System

Project Overview

FortuneTale is an intelligent personal finance tracker that transforms raw transaction data into visualized wealth stories. Designed for Chinese digital payment users, it specializes in Alipay fund analysis and daily expense tracking with automated categorization.

✨ Key Features





Alipay Integration: Auto-sync with Alipay transaction records (supporting CSV/API import)



Fund Intelligence: Real-time position analysis with yield simulation



Expense Storytelling: AI-powered spending pattern visualization



Multi-dimensional Reports: Daily/Monthly/Annual financial statements



Privacy First: All data processed locally (optional cloud sync)

Tech Stack





Frontend: Vue3 + TailwindCSS + Chart.js



Backend: Node.js + Express (optional)



Data: IndexedDB/LocalStorage + Excel export



DevOps: GitHub Actions CI/CD

📈 Why "FortuneTale"?

The name reflects our core philosophy:



"Every financial decision writes a line in your fortune's tale"
